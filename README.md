# SmartBusFlow

SmartBusFlow is a student-focused transit demo web app with a Flask backend.

## Features
- User registration / login
- Newsletter subscription (email)
- Contact form
- Schedule display
- Live map (simulated) using Leaflet
- Fare calculator
- Admin dashboard (view users, subscribers, messages)
- SQLite storage
